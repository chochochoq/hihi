- 서로 **다른 파일**을 수정했을 때
- 마자마자 수정 합시다
- 충돌
- 서로 **같은 파일**을 수정했을 때

여기서 충돌이 나는 상황은 언제일까요?

삐!

후자의 역시 또 2가지의 경우의 수로 나눠집니다.
